🗺️ Mapa de Carreira - Leandro Muniz
📖 Sobre o Projeto

Este projeto representa meu planejamento de carreira na área de tecnologia, documentando minha evolução desde os fundamentos do desenvolvimento de software até a especialização em DevOps.

O mapa foi construído com base nos meus estudos no Instituto PROA e no curso Técnico em Desenvolvimento de Sistemas do Senac Lapa Tito. Seu objetivo é servir como uma referência visual para acompanhar minha evolução profissional, identificar competências que preciso desenvolver e organizar meus próximos passos na área de tecnologia.

🎯 Objetivo Profissional

Meu objetivo é construir uma carreira sólida como Desenvolvedor Full Stack, adquirindo experiência prática em desenvolvimento web, banco de dados, APIs, arquitetura de software e soluções corporativas.

Após consolidar minha experiência como desenvolvedor, pretendo direcionar minha carreira para a área de DevOps, atuando com automação, infraestrutura, containers, cloud computing e práticas modernas de entrega contínua.
